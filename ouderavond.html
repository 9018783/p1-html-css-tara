    Visual Studio Code

    In Virsual Studio Code maken we onze website.

    we gebruiken kleine codes om alles tot een volledige website te hebben.

    Een daarvan is HTML.

    HTML is een afkorting voor Hyper Text Markup Language.Dit is het standaard taal voor het ontwerpen van websites.

    Daarnaast is er CSS.

    CSS is een nog een code om je website meer aantrekkelijk te maken. Daarin kan je kleur toevoegen. Stijl maken, de achtergrond veranderen en de breedte en hoogte van je html pagina aanpassen.

    CSS is de afkorting van Cascading Style Scheets.
